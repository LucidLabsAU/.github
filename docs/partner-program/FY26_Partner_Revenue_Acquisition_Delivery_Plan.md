#### Azure Accelerate: Factory Programs (Free MS Expert Delivery)

> ⚠️ **REAL-WORLD EXPERIENCE:** Factory programs are tedious to administer with limited practical benefit. **Recommend focusing on paid SMB programs (AI Apps, Data Platform) instead.** Only consider Factory for very complex engagements where you genuinely need MS expert augmentation AND have capacity for the administrative overhead.

**Use When:** You genuinely need Microsoft expert resources AND can handle significant coordination overhead.

| Program | Focus Area | Partner Payment | Practical Value |
|---------|------------|-----------------|-----------------|
| **Factory Gen AI Assistant** | Azure OpenAI, GenAI Solution Accelerator | $0 | ⚠️ High admin overhead |
| **Factory Analytics** | Power BI migrations, Fabric lakehouse | $0 | ⚠️ High admin overhead |
| **Factory App & Database Migration** | Windows/Linux, SQL Server, app modernization | $0 | ⚠️ High admin overhead |
| **Factory Infrastructure Migration** | VM rehost to Azure IaaS, Azure Arc | $0 | ⚠️ High admin overhead |
| **Factory Secure Azure Cloud** | Defender, Sentinel, Splunk migration | $0 | ⚠️ High admin overhead |
| **Factory Azure VMware Solution** | VMware to AVS migration | $0 | ⚠️ High admin overhead |
| **Factory VDI Migration** | RDS/Citrix to Azure Virtual Desktop | $0 | ⚠️ High admin overhead |

**Execution Window:** 200 days | **No minimum Azure spend threshold**

**Why Factory Programs Are Challenging:**
- High coordination overhead with Microsoft Factory team
- Limited flexibility in delivery approach
- Administrative burden outweighs "free" expert value
- Better to self-deliver with SMB program payments ($2K-$12K)

**When Factory MIGHT Make Sense:**
- Very large, complex migration you cannot staff internally
- Customer specifically requires MS involvement
- First-time delivery in a new technology area (learning opportunity)

**Recommendation:** Skip Factory programs. Focus on paid SMB activities (AI Apps, Data Platform) where you get direct payment and full delivery control.
