# Security Policy

## Supported Versions

We support the latest version of each repository with security updates.

## Reporting a Vulnerability

**DO NOT** create public GitHub issues for security vulnerabilities.

Please report security vulnerabilities by emailing: **notifications@lucidlabs.com.au**

Include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if available)

We will acknowledge receipt within 48 hours and provide regular updates on the status.

## Preferred Languages

We prefer all communications to be in English.
